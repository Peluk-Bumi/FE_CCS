## 📝 Konfigurasi Blockchain

### 🔹 Sepolia Testnet (Ethereum Testnet)
- Chain ID: 11155111
- RPC URL: https://sepolia.infura.io/v3/YOUR_INFURA_PROJECT_ID
- Explorer: https://sepolia.etherscan.io
- Fallback RPC: https://rpc.sepolia.org

### 🟣 Polygon Mainnet (Production)
- Chain ID: 137
- RPC URL: https://polygon-mainnet.infura.io/v3/YOUR_INFURA_PROJECT_ID
- Explorer: https://polygonscan.com
- Fallback RPC: https://rpc-mainnet.matic.quiknode.pro

### 🛠️ Cara Menggunakan
1. Buka file `.env` di FE_CCS
2. Pilih network yang ingin digunakan:
   - Uncomment bagian **Sepolia Testnet** untuk testing
   - Uncomment bagian **Polygon Mainnet** untuk production
3. Pastikan hanya SATU network yang aktif
4. Update `YOUR_INFURA_PROJECT_ID` dan `YOUR_CONTRACT_ADDRESS_HERE` sesuai konfigurasi Anda
